# 更新日志

## v1.0.2 (2026-05-05)

### 新增
- 协议编辑器新增 JSON 协议支持
  - JsonFieldEditDialog：键路径/类型/必填/默认值/枚举/最小值/最大值/正则/描述
  - JSON 字段添加/编辑/删除/上移/下移/撤销
  - JSON 数据生成、格式化、压缩、验证
  - HEX 与 JSON 协议同树管理，`[H]`/`[J]` 前缀区分
  - 切换命令时自动保存当前编辑字段
  - 鼠标悬停生成数据框显示完整内容
  - 新建协议时选择 HEX/JSON 类型
- 协议解析面板新增 JSON 解析
  - JSON 解析结果表格展示（键路径/期望类型/约束/实际值/比对状态）
  - 支持 HEX 编码的 JSON 数据自动解码
  - 支持 JSON 协议字段约束比对（类型/枚举/范围/正则）
  - 删除解析模式下拉框，自动检测 JSON/HEX
  - 选择协议对话框：全选/全不选按钮、右键折叠/展开子项、总复选框联动子项
- 自定义协议和命令掉电保存到 config.json

### 修复
- 自定义协议新建后显示 `[H]` 而非 `[J]` 的问题
- JsonFieldEditDialog 中 `add_entry_context_menu(min/max)` 传入内置函数的崩溃
- 切换自定义协议时编辑字段丢失
- `_collect_json_keys` 递归收集嵌套键导致多余「未定义」条目
- 自动解析时 `hex_clean` 奇数长度导致解码失败
- `_auto_format_input` 错误格式化 JSON 文本为 HEX

### 变更
- 删除 ui/json_viewer.py 及相关引用
- 更新 utils/version.py 版本号至 1.0.2
- JSON 预设模板必填字段设置合理的默认值
