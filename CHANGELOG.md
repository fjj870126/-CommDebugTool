# 更新日志

## v1.0.3 (2026-05-06)

### 新增
- 工具面板收藏/置顶功能（树形导航右键菜单收藏/取消收藏）
- 快捷发送模板库（导入/导出 JSON 模板文件、模板计数）
- 布局预设系统（保存/加载/删除多套窗口布局预设）
- 连接表格按协议类型颜色标记（TCP蓝/串口橙/UDP紫/WebSocket青）
- 状态栏增强：峰值速率、连接时长、智能单位显示（KB/MB）
- 日志搜索增强：正则模式（.*）、大小写敏感（Aa）切换
- 压力测试面板运行状态指示灯
- 定时发送面板运行状态指示器
- 隧道面板快捷操作（测试连接、清空日志、流量统计）
- 快捷键速查：快捷键标签页展示所有快捷键

### 优化
- MQTT 窗口分割比例从 50/50 调整为 45/55
- MQTT 面板内部布局重构：pack 改为 grid + columnconfigure，输入框自适应窗口缩放
- 主窗口分割比例保存增强：300ms+800ms 双重尝试，布局恢复更可靠
- 连接表格列宽可拖拽调整并自动保存（防抖 1 秒）
- Notebook 标签页状态持久化（快捷发送、MQTT 面板切换记忆）
- 主窗口启动速度优化（菜单栏延迟创建、配置异步加载）
- ToolTip 位置智能判断：空间不足时自动弹窗到上方

### 修复
- ToolTip 在状态栏底部时弹窗超出屏幕的问题
- ToolTip 闪烁消失问题（增加 300ms 延迟防抖）

## v1.0.2 (2026-05-05)

### 新增
- 协议编辑器新增 JSON 协议支持
  - JsonFieldEditDialog：键路径/类型/必填/默认值/枚举/最小值/最大值/正则/描述
  - JSON 字段添加/编辑/删除/上移/下移/撤销
  - JSON 数据生成、格式化、压缩、验证
  - HEX 与 JSON 协议同树管理，`[H]`/`[J]` 前缀区分
  - 切换命令时自动保存当前编辑字段
  - 鼠标悬停生成数据框显示完整内容
  - 新建协议时选择 HEX/JSON 类型
- 协议解析面板新增 JSON 解析
  - JSON 解析结果表格展示（键路径/期望类型/约束/实际值/比对状态）
  - 支持 HEX 编码的 JSON 数据自动解码
  - 支持 JSON 协议字段约束比对（类型/枚举/范围/正则）
  - 删除解析模式下拉框，自动检测 JSON/HEX
  - 选择协议对话框：全选/全不选按钮、右键折叠/展开子项、总复选框联动子项
- 自定义协议和命令掉电保存到 config.json

### 修复
- 自定义协议新建后显示 `[H]` 而非 `[J]` 的问题
- JsonFieldEditDialog 中 `add_entry_context_menu(min/max)` 传入内置函数的崩溃
- 切换自定义协议时编辑字段丢失
- `_collect_json_keys` 递归收集嵌套键导致多余「未定义」条目
- 自动解析时 `hex_clean` 奇数长度导致解码失败
- `_auto_format_input` 错误格式化 JSON 文本为 HEX

### 变更
- 删除 ui/json_viewer.py 及相关引用
- 更新 utils/version.py 版本号至 1.0.2
- JSON 预设模板必填字段设置合理的默认值
